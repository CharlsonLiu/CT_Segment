# unet、unet++

## 介绍
### 文件说明
* graph：训练过程曲线
* inputs：输入图片，手动分为train、test、val
  * JPEGImages 采用 jpg格式
  * SegmentationClass 采用 png格式
* models：存储训练结果
  * 文件夹命名格式：数据文件名+网络名+woDS(不采用深度监督)/wDS（采用深度监督）
  * config.yml：保存配置参数
  * log.csv：训练中每一轮的效果数据
  * model.pth：权重
* output：运行test后的输出

## 备注
本数据中val和test相同
